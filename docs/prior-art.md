## ② 국방 AI 선행 (포지셔닝용 2×2)

축: 구동 위치(클라우드↔온디바이스) × 지식(정적↔실시간).
- **클라우드+정적:** GenAI.mil(110만 사용자)·CamoGPT(7.5만)·**GeDAI(국방망 sLLM, 문서 40만, FS연습 실증)** — 전부 망 필요. "110만 사용자, DDIL에선 0명."
- **클라우드+실시간:** Palantir Maven(2만+ 사용자, $1.3B, NATO 도입) — "지휘관이 전장에 질문한다"는 방향성 증명. 단 기반시설 필요.
- **온디바이스+정적:** **EdgeRunner AI/WarClaw**($17.5M, 육군·AFRL CRADA, DDIL 명시, SOF 배치) — 온디바이스 군용 LLM 카테고리 검증. 단 교리를 가중치에 굽는 정적 방식.
- **온디바이스+실시간: 공백. ← 우리.**
- 문제 슬라이드 스탯: 우크라 드론 월 ~1만 대 EW 손실(RUSI), Excalibur 명중률 70%→6%(GPS 재밍), DDIL은 NATO/DoD 공식 용어.
- 금지 클레임: "최초의 군용 오프라인 LLM" (EdgeRunner 존재). 올바른 클레임: "**장교가 이미 가진 하드웨어에서 도는, 살아있는·인용 강제·이중 소스 지식베이스의 최초 결합**."

## ③ 실시간 인덱싱 구현 (핸드롤 ~150줄, 프레임워크 불요)

1. **워처:** watchdog + 경로별 threading.Timer 디바운스 2.0s (옵시디언은 타이핑 중 ~2s마다 modify 발생). `.obsidian/`·`.trash/`·`.tmp` 무시. deleted/moved → `tbl.delete(file_path=...)`.
2. **스테일 청크 규칙(최중요):** 노트가 줄면 청크 고아 발생 → 항상 **delete-then-add** (merge_insert보다 안전).
3. **스키마:** chunk_id=`relpath#i`, file_path, text, vector, source_kind(doctrine|report), report_type, report_ts, clearance. 테이블 2개(doctrine/reports).
4. **검색:** RBAC prefilter(`.where("clearance <= X", prefilter=True)`) + 보고 테이블만 반감기 리랭크 `cos + 0.25·0.5^(age_h/4)` + **최신 작전변경/중간보고 N건은 유사도 무관 무조건 주입**(데모 결정론 확보).
5. **프롬프트:** 블록마다 `[보고 <ts> | 중간보고]` / `[교범 <조항ID>]` 헤더. "보고 상충 시 최신 타임스탬프 우선, 교범은 원칙, 둘 다 인용, 상충 명시."
6. **속도:** 저장→질의 가능 ~3–4초(디바운스 2s + 임베딩 <1s + 쓰기 <0.1s). KURE-v1 상주 로드+프리웜. **UI에 지연시간 표시 = 셀링포인트.**

## ④ 스트레치: 오프라인 지도 — 조건부 GO

- **게이트:** 코어(감시→인덱스→인용 답변)가 **12–14h 시점까지 E2E 2회 완주**했을 때만 착수, **4h 타임박스** ("경로가 보고 폴리곤을 피해 재계획되면 성공"). 실패 시 슬라이드 스크린샷으로 강등. 총 8–12h짜리 작업임을 인지.
- **스택(순수 파이썬):** Geofabrik south-korea PBF(261MB, **사전 다운로드**) → osmium으로 도시 bbox 클립 → **pyrosm**(OSMnx는 Overpass API 호출 = 오프라인 함정) → networkx 가중 A*, 위협 폴리곤 교차 엣지 **가중치 ×1000 (삭제 금지** — NoPath 크래시 방지) → MapLibre+**PMTiles**(타일서버 불요) + **글리프/스프라이트 로컬 번들**(안 하면 비행기모드에서 한글 라벨 증발) + **milsymbol**(APP-6 군대부호, ~1h, 장교 심사 신뢰 급상승) + MGRS 표시.
- **정직한 프레이밍:** "도로망 기준 후보 경로 + 보고 기반 통제·위협 구간 회피" — '전술 최적 기동로' 금지(METT-TC·야지 기동 미반영을 선제 고지). ATAK+OTN(GraphHopper 오프라인 회피 라우팅)이 선행 — 우리 신규성은 **보고→위협 폴리곤→재경로 자동화 루프**뿐. 프로덕션 경로 = ATAK 플러그인/CoT 피드.
